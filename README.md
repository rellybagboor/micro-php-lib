# micro-php-lib

Micro Php Lib - Built with ❤️

## Features
- Lightweight PHP library
- Easy to integrate

## Installation
```bash
composer require micro-php-lib
```

## Usage
```php
require 'vendor/autoload.php';

$lib = new MicroPhpLib();
$lib->doSomething();
```
